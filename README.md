# Sistema de Carga ETL

Proyecto final de Lenguaje de Programacion 2 para un sistema ETL de datos inmobiliarios.

## Estructura

- `presentation/`: vista y controlador del sistema.
- `application/`: servicios de casos de uso y fachada del pipeline.
- `domain/`: entidades, enumeraciones e interfaces del dominio.
- `infrastructure/`: implementaciones concretas de repositorios, limpieza, analitica, MDM y notificaciones.
- `tests/`: pruebas automatizadas.
- `docs/`: documentacion organizada por fases PDCO.

## Fases PDCO

- `docs/01-requirements`: requerimientos y casos de uso.
- `docs/02-architecture`: arquitectura, UML y decisiones de diseno.
- `docs/03-development`: registro tecnico de implementacion.
- `docs/04-testing`: plan y resultados de pruebas.
- `docs/05-maintenance`: mantenimiento, deuda tecnica y refactorizacion.
